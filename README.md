# 📂 Daftar Project

## 1. Sistem Deteksi Gejala Social Anxiety Disorder (SAD)

### Deskripsi

Aplikasi berbasis web yang dikembangkan sebagai tugas akhir untuk membantu melakukan deteksi awal terhadap gejala **Social Anxiety Disorder (SAD)** pada mahasiswa menggunakan metode **Fuzzy Tsukamoto**.

Pengguna dapat mengisi kuesioner DASS Anxiety, kemudian sistem akan menghitung tingkat kecemasan berdasarkan aturan fuzzy sehingga menghasilkan kategori tingkat kecemasan beserta hasil diagnosis.

### Fitur

- Login Admin & User
- Dashboard Admin
- Manajemen Data Gejala
- Manajemen Rule Fuzzy
- Diagnosa SAD
- Riwayat Diagnosa
- Perhitungan Fuzzy Tsukamoto
- Hasil Diagnosa

### Teknologi

- Laravel
- PHP
- MySQL
- Bootstrap
- JavaScript
- HTML5
- CSS3

### Konsep yang Digunakan

- Fuzzy Tsukamoto
- DASS Anxiety Scale
- Authentication
- CRUD
- MVC Laravel
